# Análisis de Audiencia en Spotify: Caso Artista T4TI

## Descripción del Proyecto

*Este repositorio contiene un análisis de datos exhaustivo sobre el rendimiento del artista T4TI en la plataforma Spotify. El análisis abarca desde febrero de 2024 hasta mayo de 2026, identificando un crecimiento histórico reciente y detectando cuellos de botella clave para la expansión del artista.* 

## Métricas Clave

<img width="640" height="837" alt="Metricas Clave" src="https://github.com/user-attachments/assets/828c5eee-e782-4ae3-8254-aff31502d8fe" />

Como se observa en el resumen, el proyecto destaca por:

* 750 Streams totales con un pico histórico reciente.

* Tasa de guardados del 7.3%, lo que demuestra una alta fidelidad de la audiencia actual.

* Oportunidad: Solo 21 inclusiones en playlists, lo que indica un área crítica de mejora.
